# 科研流程图 Prompt 技能

这个技能用于把项目内容整理成可给生图工具使用的科研流程图 Prompt。

## 工作流

1. 先读项目内容：论文草稿、项目说明、模块设计、实验流程和关键结论。
2. 提取主线：核心对象、关键模块、模块关系、输入输出和最终结论。
3. 按 S-C-S-S 生成 Prompt：
   - Subject：图里要呈现的核心对象。
   - Composition：整体布局、模块关系和画面流向。
   - Structure：每个模块内部元素、箭头走向、标签和局部放大框。
   - Style：配色、边框、图标、线条、质感和期刊风格。
4. 生成科研流程图初稿。
5. 后续在 PPT 或 Adobe Illustrator 中重绘、描图和精修。

## 推荐提示词骨架

```text
请先读懂项目内容，再生成科研绘图 Prompt，不要直接泛泛描述。

项目内容：
[粘贴论文摘要、项目说明、实验流程、模块设计]

S - Subject：
[核心对象、研究系统、实验对象或流程主线]

C - Composition：
[左到右或上到下的模块化布局、模块关系、主线箭头]

S - Structure：
[每个模块内部元素、输入输出、关键变量、处理过程、结果读出、局部放大框]

S - Style：
[Nature / Cell / 工程蓝图 / 极简线稿等风格，配色、边框、图标、材质和标签要求]

负面控制：
避免低清晰度、错别字、伪科学结构、不可读标签、随机多余零件、装饰性海报风。
```

## 双控制重生成

局部图高清重生成时，建议使用两类参考：

- 线稿图：约束结构、轮廓、管路、箭头和选区边界。
- 色稿图：约束材质、配色、光影和整体质感。

重生成要求：只增强目标区域，不改变整体科学含义，不添加无关零件。
